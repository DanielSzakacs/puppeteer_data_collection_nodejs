# TennisAbstract scraper

Run:

```bash
node scrape-tennisabstract.js players.txt output.csv
```

The generated CSV includes the following columns:

- `Player`
- `Age`
- `Plays`
- `Date`
- `Tournament`
- `Rd`
- `Rk`
- `vRk`
- `DR`
- `A%`
- `DF%`
- `BPSvd`
- `Opponent`
- `Winner`

node scrape-tennisabstract.js players.txt output.csv
